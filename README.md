# Rawlyn Rogers Website

Modern full-stack personal portfolio website built using Spring Boot, React, and Maven with scalability, security, and AWS deployment in mind.

## Tech Stack

### Backend
- Java
- Spring Boot
- Maven
- REST APIs

### Frontend (Planned)
- React
- JavaScript
- HTML/CSS

### Infrastructure Goals
- AWS EC2
- HTTPS
- Nginx Reverse Proxy
- Secure Environment Variable Management

## Planned Features

- Responsive homepage
- Expandable project showcase
- Resume downloads
- Contact integrations
- GitHub and LinkedIn integration
- Secure backend architecture

## Development Goals

- RESTful API architecture
- Secure credential handling
- Scalable cloud deployment
- CI/CD integration
- Docker containerization
